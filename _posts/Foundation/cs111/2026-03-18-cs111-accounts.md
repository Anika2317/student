## FA1 — SkyKingdom Environment {#fa1}

FA1 introduced the OCS GameEngine. I built the SkyKingdom environment and learned how Background and Barrier objects behave inside the Game Loop.

### What I Built
- Configured layered background assets for the SkyKingdom level
- Defined `hitbox` properties for all Barrier objects using **Data-Driven Design**
- Verified the engine's `update()` cycle correctly repositioned objects each frame

### Key Code: The Game Loop
```javascript
// Iteration over all game objects every frame
gameObjects.forEach(obj => {
    obj.update(); // recalculates position — Mathematical operators
    obj.draw();   // renders to canvas — Canvas I/O
});
```

## FA2 — Interaction & NPC Design {#fa2}

FA2 added interaction mechanics. I implemented NPC behavior using reaction functions triggered by player proximity and state.

### What I Built
- Wolf NPC with configurable `hitbox.range` — JSON Object config
- `reaction()` method triggered by proximity — Boolean Expression
- Tuned NPC behavior via **GameBuilder Visual Interface** — Data-Driven Design

### Key Code: Detection vs Reaction
```javascript
// Nested conditions — CS111 rubric requirement ✅
handleCollision(other, direction) {
    if (other.type === "Player") {              // condition: type check
        if (this.distanceTo(other) < 50) {     // nested: proximity
            if (player.hasItem("basket")) {    // nested: inventory
                this.reaction("friendly");
            } else {
                this.reaction("hostile");
            }
        }
    }
}
```
