# 🛠️ Tools Setup
Follow these steps to get your computer ready:

1. **VS Code:** Install the editor and the "Python" extension.
2. **Homebrew (Mac):** Essential for installing Ruby and Jekyll.
3. **Python Venv:** Run `python3 -m venv venv` in your project folder.

> **Tip:** If you see a "Module Not Found" error, make sure your virtual environment is active!
